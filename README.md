# Equation-Driven Pots

A generative design project that creates customizable, 3D-printable, multi-color pots driven by mathematical equations.

## About the Project
This repository contains the core development files, Python scripts, and logic used to generate parametric 3D models of pots. By adjusting underlying mathematical equations, users can dictate the form, texture, and multi-color segmentation of the final 3D model, preparing it seamlessly for 3D printing.

This main repository is intended for developers, contributors, and anyone interested in the underlying generative design workflow.

## Features
* **Equation-Based Modeling:** Generate complex, organic, or geometric pot shapes using math.
* **Multi-Color Support:** Specifically designed to segment models for multi-color 3D printing.
* **Export Ready:** Generates files optimized for modern 3D slicers.
