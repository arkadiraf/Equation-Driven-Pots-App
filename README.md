# Equation-Driven Pots App

This repository hosts the **online development tools** for the Equation-Driven Pots project.

## Main recommended GUI

The current main browser interface for the project is the unified:

**[Equation Driven Pot Designer](https://arkadiraf.github.io/Equation-Driven-Pots/)**

This is the newest all-in-one GUI and the recommended place to start. It brings together the current workflow in a single interface, including:

- base scaffold design
- structural geometry
- surface texture
- dual pattern color layers
- field modifier experiments
- field distortion experiments
- pot, plate, and pot+plate workflows
- STL and multi-part 3MF export

## Development tools hosted here

This repository still matters because it hosts some of the earlier and more experimental browser tools for online use:

**[Legacy / development app collection](https://arkadiraf.github.io/Equation-Driven-Pots-App/)**

These tools remain useful for:
- testing ideas in smaller focused interfaces
- preserving earlier workflows
- comparing older and newer approaches
- exploring experiments that may later be folded into the unified GUI

So the current relationship is:

- **Equation Driven Pot Designer** = main recommended unified GUI
- **Equation-Driven-Pots-App** = hosted collection of development and legacy browser tools

## About the project

Equation-Driven Pots is a generative design project for creating customizable, 3D-printable forms from mathematical equations.

Depending on the tool, users can control:

- overall form
- surface texture
- color segmentation
- nonlinear field behavior
- field modifiers
- field distortion
- structural pot and plate geometry

The project is built around the idea that equations are not just a way to describe a finished object, but a direct design medium.
